Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n community nodes documentation.

## Operations

- **Crawl** — Crawl a website and extract content from multiple pages
- **Scrape** — Scrape content from a single page
- **Search** — Search the web and optionally retrieve full page content
- **Links** — Extract all links from a page
- **Screenshot** — Capture a screenshot of a page
- **Transform** — Fetch a URL and convert its content to another format (markdown, HTML, text, XML)
- **Get Credits** — Check your available Spider API credits

All content operations support configurable return formats (Markdown, HTML, Plain Text, XML) and advanced options including anti-bot bypass, headless browser rendering, proxy rotation, and readability mode.

## Credentials

You need a Spider API key to use this node.

1. Sign up at [spider.cloud](https://spider.cloud/?utm_source=n8n&utm_medium=community_node&utm_campaign=n8n-nodes-spider)
2. Go to [spider.cloud/api-keys](https://spider.cloud/api-keys?utm_source=n8n&utm_medium=community_node&utm_campaign=n8n-nodes-spider) to generate your API key
3. In n8n, create a new **Spider API** credential and paste your key

## Compatibility

Tested with n8n version 1.x. No known incompatibilities.
